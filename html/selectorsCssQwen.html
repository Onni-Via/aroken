Демонстрация всех видов селекторов CSS

Универсальный селектор (*)

Определение: Универсальный селектор применяется ко всем элементам на странице. Он используется для глобальных стилей.

Этот текст будет затронут универсальным селектором.

Селектор по тегу (например, p)

Определение: Селектор по тегу применяет стили ко всем элементам указанного типа (например, параграфам).

Этот параграф стилизован с помощью селектора по тегу.

Селектор по классу (.class)

Определение: Селектор по классу применяет стили к элементам с указанным классом. Классы используются для группировки элементов.

Этот блок стилизован через селектор по классу.

Селектор по идентификатору (#id)

Определение: Селектор по идентификатору применяет стили к уникальному элементу с указанным ID. Идентификатор должен быть уникальным на странице.

Этот блок стилизован через селектор по идентификатору.

Дочерний комбинатор (селектор1 > селектор2)

Определение: Дочерний комбинатор применяет стили только к непосредственным дочерним элементам указанного родителя.

Этот параграф — дочерний элемент родителя.

Смежный соседский комбинатор (селектор1 + селектор2)

Определение: Смежный соседский комбинатор применяет стили к элементу, который следует сразу за указанным элементом.

Общий соседский комбинатор (селектор1 ~ селектор2)

Определение: Общий соседский комбинатор применяет стили ко всем элементам, которые следуют за указанным элементом (не обязательно сразу).

Первый параграф.

Промежуточный элемент.

Третий параграф, который является общим соседом первого.

Атрибутный селектор ([attr])

Определение: Атрибутный селектор применяет стили к элементам, у которых есть указанный атрибут (или его значение).

Ссылка с атрибутом title.

Псевдокласс :hover

Определение: Псевдокласс :hover применяет стили к элементу, когда пользователь наводит на него курсор мыши.

Псевдокласс :first-child

Определение: Псевдокласс :first-child применяет стили к элементу, который является первым дочерним элементом своего родителя.

  • Первый элемент списка.
  • Второй элемент списка.

Псевдокласс :nth-child(n)

Определение: Псевдокласс :nth-child(n) применяет стили к элементу, который находится на указанной позиции среди своих соседей.

  • Первый элемент списка.
  • Второй элемент списка.
  • Третий элемент списка.

Псевдокласс :focus

Определение: Псевдокласс :focus применяет стили к элементу, который находится в фокусе (например, поле ввода).
